# 主菜单设计文档

## 树状结构图

```
主界面
├── 开始游戏
│   └── 选择存档位（三个）
│       └── 如有存档是否覆盖？
│           ├── 否 → 返回选择存档位
│           └── 是 → 切换场景（进入游戏）
├── 画廊
│   └── 切换到画廊场景
│       ├── 过场动画和CG浏览
│       └── 图鉴（游戏内元素介绍）
├── 设置
│   ├── 窗口分辨率
│   ├── 全屏幕
│   ├── 音乐音量
│   ├── 音效音量
│   ├── 语言
│   └── 游戏按键教学（不提供改键）
└── 退出游戏
```

## 详细说明

### 1. 开始游戏
- **功能**：启动游戏流程
- **子流程**：
  - 显示三个存档槽位供玩家选择
  - 若选中的存档槽已有存档，提示是否覆盖
    - 选择"否"：返回存档选择界面
    - 选择"是"：创建新存档并切换至游戏场景

### 2. 画廊
- **功能**：查看已解锁的游戏内容
- **子功能**：
  - **过场动画和CG浏览**：播放已解锁的剧情动画和插画
  - **图鉴**：查看游戏内元素介绍（敌人、道具、场景等）

### 3. 设置
- **功能**：调整游戏各项参数
- **可配置项**：
  - 窗口分辨率：调整游戏窗口大小
  - 全屏幕：切换全屏/窗口模式
  - 音乐音量：调节背景音乐音量
  - 音效音量：调节游戏音效音量
  - 语言：切换游戏显示语言
  - 游戏按键教学：显示操作说明（不可修改按键）

### 4. 退出游戏
- **功能**：关闭游戏程序并返回操作系统

## 界面返回逻辑

- 所有子界面（设置、画廊）可通过返回按钮回到**主界面**
- 存档选择界面可通过返回按钮回到**主界面**
